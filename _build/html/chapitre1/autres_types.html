Autre types de variables#

from IPython.display import display, Markdown

Variable None#

La variable None ne prend aucune autre valeur qu’elle même constitue un type de variable à elle toute seule. Le type None est distinct de tous les autres types de variables. L’utilité d’une telle variable se trouve dans la gestion d’exceptions

var = None
print ('The condition "var == None" is', (var == None))
print ('The condition "not var" is', (not var))

The condition "var == None" is True
The condition "not var" is True

Booléens#

Les variables bouléennes premettent de représenter les résultats des opérations dans l’algèbre de Boole (ou opérations logiques). Elles peuvent prendre les deux valeurs True ou False.

x = True
print (type (x))
print (f'La valeur de x est {x}')
print (f'La valeur de x considérée comme un entier est {x:d}')

<class 'bool'>
La valeur de x est True
La valeur de x considérée comme un entier est 1

Fonction all ()#

table = '|a|b|c|all (a,b,c) |\n|:-:|:-:|:-:|:-:|\n'
for a in [True, False]:
    for b in [True, False] :
        for c in [True, False] :

Chaînes de caractères (première partie)#

Introduction minimale#

Les variables figurant à l’intérieur d’une paire d’apostrophes (’) ou de guillemets (”) sont interprétées comme des chaînes de caractères. Le type d’une chaîne de caractères est str.

Formattage des nombres convertis d’une chaîne de caractère#

A l’intérieur d’une chaîne de caractère, il est permis d’évaluer la valeur d’une variable numérique et de la convertir en chaîne de caractère. La variable numérique doit figurer entre accolades et le format de représentation doit être indiqué derrière deux points (:).

s = 'Coucou'
print (s)
print ('le type de s est : ', type (s))

Coucou
le type de s est :  <class 'str'>

Python permet la concaténation de chaînes de caractères par simple utilisation des opérations + et * :

s = 'Coucou'
s2 = s + s
print (s2)

CoucouCoucou

s = 'Coucou'
s3 = s * 3
print (s3)

CoucouCoucouCoucou

# formattage d'un entier
n = 2
print (f'la valeur de l\'entier n est {n:d}')

x = 23.456
print (f'la valeur du réel x est {x:f}, avec 6 décimales, par défaut')
print (f'la valeur du réel x est {x:.2f}, arrondis avec 2 décimales')
print (f'la valeur du réel x est {x:.2e}, en notation ingénieur')

la valeur de l'entier n est 2
la valeur du réel x est 23.456000, avec 6 décimales, par défaut
la valeur du réel x est 23.46, arrondis avec 2 décimales
la valeur du réel x est 2.35e+01, en notation ingénieur

s = 'Coucou'
print (s[0])

C

Les chaînes de caractères sont non modifiables.

s = 'Coucou'
s [0] = 'D'

Toute modification d’une variable de type str nécessite d’écraser sa valeur précédente :

s = 'Coucou'
print ('Avant modification, la chaîne s contient : ', s)
s = 'Doucou'
print ('Après modification, la chaîne s contient : ', s)
