Quel équipement pour le PH641 ? #

Tous les exercices étant sur machine, il est obligatoire de venir en cours muni d’un ordinateur, le vôtre ou un cartable numérique. L’interpréteur Python peut-être installé sur les systèmes d’exploitation Linux, Windows et MacOS. Le département de mathématique de l’UVSQ met à disposition un interpréteur en ligne accessible avec les identifiants UVSQ : jupyter.ens.uvsq.fr

Pour ce cours, il n’est donc pas indispensable d’installer Python sur son propre équipement, mais une installation locale est fortement recommandée pour le confort de travail.

Installation sous Linux :#

  • Installer notebook :

 pip install notebook

  • Installer l’extension nbextensions de notebook:

 pip install nbconvert 
 pip install jupyter_contrib_nbextensions

  • Activer l’éditeur convivial des extensions de notebook :

 jupyter nbextensions_configurator enable

  • Lancer un notebook de jupyter :

 jupyter notebook
